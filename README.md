# Yoga Pose Detection  

## Project Overview  
Built a computer vision based Yoga Pose Detection system that classifies human poses from images using deep learning. Demonstrates practical application of CNNs, image preprocessing, and model evaluation.

---

## Key Highlights  
- Developed an image classification model for multiple yoga poses  
- Applied data preprocessing and augmentation to improve model generalization  
- Implemented an end-to-end ML pipeline: data → training → evaluation → prediction  

---

## Tech Stack  
- Python  
- TensorFlow / Keras  
- OpenCV  
- NumPy, Pandas  

---

## Approach  
- Preprocessed dataset (resizing, normalization, augmentation)  
- Built and trained a Convolutional Neural Network (CNN)  
- Evaluated model using accuracy and loss metrics  
- Generated predictions on unseen data  

---

## Results  
- Model successfully classifies yoga poses with strong performance on test data  
- Demonstrates understanding of deep learning workflows and model optimization
